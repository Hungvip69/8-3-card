# Asset notes

- `floral-bg.svg`: Floral illustration used as the visual background accent.
- `sample-music.wav`: Local loopable melody used by default in the music toggle.

To replace assets:
1. Put your own image/audio files in this folder.
2. Update `ASSETS.bgImage` and `ASSETS.musicFile` in `main.js`.
3. Keep file sizes compact for better mobile performance.
